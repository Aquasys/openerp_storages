Development environment
=======================

Git repo
--------

Initialize Git flow:

.. code-block:: console

    $ git flow init

Add git notes (cf http://doc.erp.aquasys.co.jp/Usage/1_git.html#notes)


Code review
-----------

In `Barkeep <http://review.aquasys.co.jp/commits>`_, add the following filter::

    repos:openerp_storages branches:all
